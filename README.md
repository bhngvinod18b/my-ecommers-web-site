# 🛒 Django E-Commerce Website

A full-stack E-Commerce web application built using Django.  
This project includes user authentication, product management, customer dashboards, booking/orders, and manager/admin functionality.
this project is about inventery management with stock check and booking products manager can acces all ordels list and update status like pending, cancel, conformend

## 🚀 Features

### 👤 Authentication
- User Registration
- User Login & Logout
- Role-based access (Customer / Employee / Manager)

### 🛍 Product Management
- Add Products
- Product Categories
- Product Quantity & Price Management
- Product Search & Filtering

### 📦 Order System
- Place Orders
- Booking Management
- Order Status Updates
- Stock Quantity Updates Automatically

### 📊 Dashboard
- Customer Dashboard
- Manager Dashboard
- View Orders
- Total Amount Calculation

### 🔒 Security Features
- Login Required Protection
- Atomic Transactions for Safe Ordering
- Role-based Redirection
